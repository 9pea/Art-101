Project 1

I did not have a plan going into this project. My goal was to just experiment with the code and create brushes that were distinct and unique from each other. The sketching program consists of four unique brushes, an eraser brush, a canvas reset button, and a screenshot button.
What I found successful was finding an aesthetic that would work for the brushes which I believe is a more abstract style that involves a lot of movement. I am proud of myself for getting the increase/decrease value switch to work which I feel adds more functionality to this drawing program.
What I found challenging was getting the brush codes to work in the first place. I would say that my coding knowledge is still limited so trying to get those codes to work in unison while also trying to make brushes that looked uniqued was a good challenge. Additionally, I also had a hard time with figuring out how to stop the elements of one brush affecting the other, such as the stroke for one affecting the stroke for the one that gets used after, but I am glad that I was able to fix that issue.
