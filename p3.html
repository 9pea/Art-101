Exercise 5

For this assignment, my teammates were Jupiter, Nabeeha, and Jingyi. For our them we wanted to have it be broad so we focused on making charcters. I took Nabeeha's and Jingyi's number gridarrays but I did not take Jupiter's because they had too many colors. I made 8 custom functions to remap my 2 arrays. How they remap the images: maps the arrays with a color palette; changes the color palette of the arrays; makes the arrays monochrome; changes the shapes that make up the arrays; mapped the arrays using the images; make the images tinted; maps the textarray with a color palette; and changes the color palette for the textarray. My biggest struggle for this project was working on pages 4 and 5 simply because of how long it took to load the page on my laptop.
I tried to utilize the following concepts: making 2 distinct GridArrays with each being 16x16; having 1 array using number values and the other using text strings; mapping the GridArray using a bitmap image; mapping and visualizing each array twice with 2 distinct versions; and having at least 4 png/jpeg images and making a mapped image based on my data.

Project 3

For Project 3, I wanted to base my clock around the theme of emotions. This semester, or just this whole year really, was kind of emotionally draining which inspired me to base this idea off of it. The clock will have hours, minutes, seconds, and milliseconds. Each hour that passes, text will appear to indicate what activity I am doing at that time of day. In order to indicate the time of day, the background and ellipse will change colors.
I tried to apply the following concepts: 12+ based hour cycle, real-time passage of time, 4 time divisions, visual representation of time, use of millis, hour, minute, and second, and use of Map function.
